The basic type provided by this package is ARCHModel. It implements the interface of StatisticalModel from StatsBase. An instance of this type contains a vector of data (such as equity returns), and encapsulates information about the volatility specification (e.g., GARCH or EGARCH), the mean specification (e.g., whether an intercept is included), and the error distribution.
